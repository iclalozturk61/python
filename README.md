# py-GeneralExamples-v1
genel basit anlama uygulamalarıdır

Python dosyalarını Windowsta not defteri aracılığıyla yazıp çalıştırmak hakkında: 

1- yeni bir metin belgesi açıp içine normal kod yazarmış gibi yazıyorsun

2- kaydederken .py olarak kaydediyorsun (isimlendirmesi bu şekilde yapılıyor)

3- komut istemine girip, cd Desktop yazarak dosyanın olduğu klasöre giriyorsun

4- python kod.py yazarak belgeyi çalıştırıyorsun (bu örnekte doyanın ismi kod)
